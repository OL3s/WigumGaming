# WigumGaming

Backup repository for essential WigumGaming files that are not already covered by separate project repositories.

Tracked here:

- command notes in `BashCommands/`
- shared or backup images in `Images/`

Ignored here:

- `BloggStorage/`
- `Godot/`
- `Webpages/`

Those folders contain separate Git repositories and are backed up through their own GitHub remotes.
